Intel® QuickAssist Technology Documentation Hub

This documentation serves as the central repository for Intel® QuickAssist Technology (Intel® QAT) implementation, deployment, and optimization guidelines. Whether you’re a developer, system administrator, or performance engineer working with hardware acceleration technologies, this resource provides the essential information you need to successfully implement, deploy, and optimize Intel QAT solutions for cryptographic and compression workloads.

What You’ll Find Here

This documentation covers:

  • Legal & Overview: Essential legal notices, introduction to Intel QAT technology, and key terminology.

  • Release Notes: Platform-specific release information for In-Tree, Linux, and VMware implementations.

  • User Guides: Comprehensive guides covering getting started, programming, virtualization, performance optimization, and QATlib usage.

  • Developer Resources: Application notes and API programming guides for advanced Intel QAT integration.

  • Support & Resources: Contact information, additional documentation, and community resources.

Getting Started

If you’re new to Intel QAT, we recommend starting with:

  1. Introduction - Understanding Intel QAT technology and its benefits.

  2. Getting Started Guide - Your first Intel QAT implementation.

  3. Terminology - Key concepts and definitions for Intel QAT development.

For experienced users looking to optimize performance or implement advanced features, jump directly to the Performance Optimization Guide or API Programmer’s Guide.
