Build Containers with Intel® QAT Software

Note

If not running as root user, sudo will need to be added to the Linux* instructions below.

  1. Copy the repository.

    git clone https://github.com/intel/intel-device-plugins-for-kubernetes

  2. Build the image.

    cd intel-device-plugins-for-kubernetes
make openssl-qat-engine
