.. Notes to generate nice TOC for PDF documents
..
.. To generate a PDF of a single document (e.g. GSG only, RN only, PG only, etc.),
.. comment out all unneeded sections in the below index and leave only the one you 
.. want to generate the PDF for.
..
.. To generate a nice TOC for PDFs (multiple chapters), the content from the respective
.. index.rst section must be copied to this master index.rst file. For example, to generate
.. a nice TOC for the GSG, the following index content in this master index.rst file:
..       GSG/GSG2/index
.. must be replaced with the following content from the GSG index.rst file:
..       GSG/GSG2/introduction
..       GSG/GSG2/systemconfiguration
..       GSG/GSG2/installation
..       GSG/GSG2/sampleapplications
.. This will make the PDF output to have multiple chapters for each of the sections in the document
.. rather than only one chapter for the entire document.
